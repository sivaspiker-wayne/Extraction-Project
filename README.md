# Data Extraction Project

**How to contribute to this project?**

There are 3 text file and contributors should write functions to extract
essential information like PAN, Aadhaar number, credit card number, mobile number, email ID and so on.

Each contributor should work on only one function at a time. Once test on 
contributor's end, they should create a Pull Request (PR).

Use Python's `re` module for this project. No other language should be used.
